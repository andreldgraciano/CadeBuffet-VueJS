v1/search_buffet/nomeBuffet

Para filtrar buffets pelo nome

api/v1/events/1?date=2024-10-27&amount_people=150

Para verificar disponibilidade de um evento, passando id do buffet, parametro date, e amount_people

Possiveis respostas:

  • {status: "Unavailable for today or old dates"}
  • {status: "Unavailable for date #{date}"}
  • { status: 'Available', event_name: event.name, date: param_date, amount_people: param_amount_people, total_value: total_value }
  • {status: 'Event unavailable for this number of people', min_people: event.min_people, max_people: event.max_people}
